Euclidean Distance Problem

    Performed euclidean distance algorithm on self-made registry of locations. Locations are in GPS coordinates in diffrent
        formats. Degree, minutes, and seconds; which is a standard. This format is then converted to decimal degree notation
        to ease the computation resources that will be used. After the algorithm completes it makes a new registry with one location
        directed to the shortest path available. Visualizations are also performed on the location using libraries -- seaborn and plotly.
